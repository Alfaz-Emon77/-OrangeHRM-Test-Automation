
# OrangeHRM Test Automation (Maven + Selenium + TestNG)

## Project Overview
This project demonstrates **automation testing of the OrangeHRM demo site** (https://opensource-demo.orangehrmlive.com/) using **Selenium WebDriver, TestNG, and Maven**.  
The project covers login, employee management, and other HR workflows as part of QA portfolio/demo.

> ⚠️ **Note:** This project is for **educational/demo purposes** only. No live credentials are used.

---

## Tools & Technologies
- **Selenium WebDriver** – Automates browser actions
- **TestNG** – Test execution and reporting
- **Maven** – Project build and dependency management
- **Java** – Programming language
- **CSV/Excel** – Data-driven testing
- **Git/GitHub** – Version control and portfolio showcase

---

## Automated Tasks / Test Scenarios
1. **Login Flow**
   - Open OrangeHRM demo site
   - Enter username (`Admin`) and password (`admin123`)
   - Click Login
   - Verify Dashboard welcome message

2. **Add Employee**
   - Navigate to PIM → Add Employee
   - Fill in employee details
   - Save and verify employee creation

3. **Search Employee**
   - Navigate to Employee List
   - Enter employee name or ID
   - Click Search and verify results

4. **Other Possible Tasks (Optional)**
   - Logout flow
   - Edit employee details
   - Delete employee
   - Upload employee photo
